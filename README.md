# ניווט חופי — מתרגל ימאות 🧭

אפליקציית תרגול לכללי מניעת התנגשויות בים (COLREG 1972) לקראת בחינת **משיט 30** (ימאות).
כוללת: אורות וצורות של כלי שיט (יום/לילה), מצופים (IALA אזור A), אותות קול,
תרחישי זכות קדימה אינטראקטיביים, וחידון עם **שאלות שנוצרות אוטומטית** מהתקנות (אינסוף תרגול).

> כל הנתונים אומתו מול הטקסט הרשמי של COLREG (USCG NavCen) ומול IALA R1001.

## ▶️ איך מפעילים

### במחשב
לחצו לחיצה כפולה על `index.html`. זהו. עובד **לא מקוון** (אין צורך באינטרנט או שרת).

### בטלפון (כמו אפליקציה)
1. העבירו את שני הקבצים (`index.html` + `rules.js`) לטלפון — באותה תיקייה (אימייל / Drive / AirDrop).
   > חשוב: שני הקבצים חייבים להיות **יחד באותה תיקייה**.
2. פתחו את `index.html` בדפדפן (Safari / Chrome).
3. **הוספה למסך הבית:**
   - **iPhone (Safari):** כפתור שיתוף → «הוסף למסך הבית».
   - **Android (Chrome):** תפריט ⋮ → «הוסף למסך הבית».
4. עכשיו יש אייקון שנפתח במסך מלא כמו אפליקציה.

## ✏️ איך מעדכנים תקנה (עריכת `rules.js`)

פותחים את `rules.js` בכל עורך טקסט. בראש הקובץ יש **מדריך עריכה מלא** עם הסבר לכל שדה.
לכל פריט יש שדה ייחוס:

```js
ref: { colreg: "Rule 27", israeli: "" }
```

מלאו את `israeli` בייחוס לתקנה הישראלית המתאימה כדי שתוכלו להצליב מול המקור הרשמי.

### דוגמאות נפוצות

**להוסיף כלי שיט חדש** — העתיקו אובייקט קיים מתוך `vessels[]`, שנו `id` ואת האורות/הצורות:
```js
{
  id: "my_new_vessel",
  name: "שם בעברית",
  shortName: "שם קצר",
  ref: { colreg: "Rule XX", israeli: "תקנה ..." },
  giveWayClass: "power",          // power|sail|fishing|ram|nuc|cbd|anchor|aground|towing|pilot
  underway: true,
  lights: [ { place:"masthead", color:"white", arc:225, label:"...", desc:"..." } ],
  shapes: [ { shape:"ball", place:"stack", color:"black", label:"...", desc:"..." } ],
  summary: "...",
  examTip: "..."
}
```
ערכי `place` של אורות, ערכי `shape` של צורות, וכל השאר — מפורטים בראש `rules.js`.

**להוסיף אות קול / מצוף / תרחיש** — אותו עיקרון: העתיקו פריט קיים מתוך
`sounds[]` / `marks[]` / `scenarios[]` ושנו את הערכים.

> אחרי כל עריכה — שמרו ורעננו את הדף. אם משהו לא מופיע, כנראה יש פסיק חסר ב‑`rules.js`.

## 🧠 איך נוצרות השאלות

מחולל השאלות בוחר עובדה אקראית מהנתונים, ושואב את **התשובות השגויות מעובדות אמיתיות אחרות**
באותו נושא. כך כל שאלה נוצרת מחדש אך תמיד **עובדתית ונכונה**, והתשובה הנכונה תמיד מאומתת מול התקנות.
המחולל גם נמנע אוטומטית משאלות דו‑משמעיות (למשל: כלי שיט ממונע וכלי שיט במפרש+מנוע מראים אותם
אורות בלילה — לכן הם לא יוצגו כשתי תשובות לזיהוי לפי אורות).

סוגי שאלות: זיהוי לפי אורות/צורות · ריקול אורות · ריקול צורות · זכות קדימה (תקנה 18,
חזית, חיתוך, עקיפה, מפרשיות) · אותות קול · מצופים.
