# ימאות 🧭 — לומדת ההכנה לבחינות משיט

אפליקציית תרגול מקיפה לבחינות ההסמכה למשיטים (משיט 11/12/13/30) של רשות הספנות והנמלים:
תקנות מניעת התנגשויות (COLREG 1972), אורות וצורות, מצופים (IALA אזור A), אותות קול, דגלי קוד,
תרחישי זכות קדימה אינטראקטיביים, **מאגר השאלות הרשמי המלא (1,399 שאלות)**, סימולטור מבחן
אמיתי (כולל שאלות פוסלות!), **ניווט חופי** עם מפת אימונים אינטראקטיבית, ומציג **תלת־ממד**
של אורות כלי שיט מכל כיוון.

> 🌐 גרסה חיה: `https://itaybracha31.github.io/Yamaut/`
> 📱 עובד מצוין בטלפון — הוסיפו למסך הבית וקבלו אפליקציה מלאה (כולל עבודה לא מקוונת).

## ✨ מה יש בפנים

| מדור | תוכן |
|------|------|
| 🏠 **בית** | דרגה ימית (XP), רצף ימים, יעד יומי, שליטה בנושאים, תגים |
| 📚 **לימוד** | **אורות וצורות בתלת־ממד** — סובבו כלי שיט וראו אילו אורות וצורות נראים מכל כיוון, ביום ובלילה (מנוע תלת־ממד קטן משלנו) · מצופים (IALA אזור A) עם מקצבי ההבהוב האמיתיים · אותות קול מושמעים · דגלי הקוד הבינלאומי |
| 🎯 **תרגול** | חידון אינסופי שנוצר מהתקנות · **תרחישים** — גררו את כלי השיט וכיוונו את התמרון, ואז «בדוק תמרון» מריץ סימולציה פיזיקלית שבודקת אתכם לפי התקנות. תרחיש חדש נחשף בסריקת מכ"ם עדינה, **משקפת** בפינת הלוח מציגה את הפתרון, וחץ **איפוס** מאפס את התמרון. אפשר «המשך אוטומטית» — קפיצה לתרחיש הבא מיד אחרי תמרון מושלם. כולל **מצב לילה** — זיהוי לפי אורות בלבד · חזרה חכמה על טעויות |
| 🎓 **מבחן** | סימולציית המבחן האמיתי מתוך **המאגר הרשמי של משרד התחבורה** — לפי מסלול (משיט 30 ימאות / נווט מכשירים / מכונאות, משיט 13/12/11), עם טיימר, ציון עובר, ו**שאלות פוסלות** כמו במבחן האמיתי |
| 🗺️ **ניווט** | מפת אימונים אינטראקטיבית — קביעת מקום (Fix) משני כיוונים, כיוון־ומרחק, ניווט שולחני (DR) · מחשבוני מהירות/זמן/מרחק ו‑TVMDC · כרטיסיות מושגי ניווט |

הכול נשמר מקומית (localStorage) — התקדמות, XP, תגים, רצף ימים, היסטוריית מבחנים ושיאי ניווט.

**⚙️ הגדרות** (מהכפתור בתחתית המסך): מצב תצוגה **בהיר / כהה / אוטומטי** (לפי המכשיר), צלילים, רטט,
יעד XP יומי (30/50/100), ואיפוס התקדמות.

**🔄 עדכון חי של מאגר השאלות:** מתוך ההגדרות אפשר «למשוך עדכון» — האפליקציה שולפת את המאגר העדכני
ישירות מ‑data.gov.il (רשות הספנות והנמלים) ומחליפה את המאגר המובנה, עם אפשרות «שחזר למובנה».
תקנות ותכנים אחרים מתעדכנים עם גרסת האפליקציה (Service Worker) בכפתור «בדוק עדכון».

## 📌 מקורות ואמינות

- COLREG 1972 — הטקסט הרשמי (USCG NavCen) + תקנות הנמלים (מניעת התנגשויות בים), תשל"ז‑1977.
- IALA Maritime Buoyage System (R1001) — **אזור A** (ישראל).
- **מאגר השאלות הרשמי**: רשות הספנות והנמלים, data.gov.il (dataset `opercraft_tq` → 1,399 שאלות פעילות), נשלף 2026‑07‑03. ניתן למשוך עדכון חי מתוך ההגדרות. דף המאגר: gov.il ‏«מאגר שאלות למבחנים הסמכה למשיטים».
- מבנה הבחינות לפי **סילבוס משיטים 2026** הרשמי (בתוקף מ‑01.01.2026).
- אחוזי המעבר במבחנים אינם מפורסמים רשמית — באפליקציה מוצגים ערכים לפי דיווחי בתי ספר לחובלים.

הנתונים נבדקו מול המקורות הרשמיים; באחריות המשתמש להצליב מול הפרסומים העדכניים של
רשות הספנות והנמלים. בהצלחה בבחינה! ⚓
